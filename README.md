# Social Media Platform
Develop a social media platform that includes user authentication, news feed, posts, comments, likes, notifications, and additional features such as photo and video uploads, stories, and friend requests.

## What to do?
### User Authentication

Registration and Login:

Implement registration and login functionalities for users.

Include options for password reset and change.

Ensure secure user authentication and session management.

## Core Features
 ### News Feed:
Create a news feed that displays posts from users and their friends.

Implement algorithms to prioritize and sort posts.

### Posts:
Allow users to create and publish posts.

Support text, photo, and video content in posts.

Enable editing and deletion of posts.

### Comments:
Allow users to comment on posts.

Implement functionalities to edit and delete comments.

### Likes:
Implement a like button for posts and comments.

Display the number of likes and who liked the post.

### Notifications:
Create a notification system to alert users about interactions such as likes, comments, friend requests, and mentions.

Display a notification feed and individual notification details.

### Photo and Video Uploads:
Allow users to upload photos and videos to their posts and profiles.

Implement functionalities to manage and delete uploaded media.

### Stories:
Enable users to create and view stories that disappear after a specified time (e.g., 24 hours).

Implement functionalities to add photos, videos, and text to stories.

### Friend Requests:
Implement a system for users to send, receive, and manage friend requests.

Create functionalities to accept or decline friend requests.

### User Profiles
Profile Management:
Allow users to view and edit their profile information.
Implement profile picture and cover photo uploads.
Include profile details such as bio, location, and website links.

### Friend List:
Display a list of friends on the user's profile.
Implement functionalities to view friend profiles and interact with them.

## Techstacks needs to be used : 

MERN stack
TailwindCSS for styling
